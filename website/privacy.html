Privacy

Project Aegis is a prototype. By design, it minimizes data export. The default tools provided here do not exfiltrate user data.

Local-first

Scanning and analysis run locally in native code. Any upload of telemetry or artifacts should be opt-in and audited.

Open audit

Engine code is intended for audit. We recommend code review of the FFI boundary and packaging scripts before production use.

For audits and contributions, see the repository README.
