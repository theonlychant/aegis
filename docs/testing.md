Testing matrix and device targets

Tiers:
- Tier 1: iOS 26+ (modern) — full feature tests (URL filtering, NE, App Attest)
- Tier 2: iOS 18-25 — standard features (network filtering, scanning)
- Tier 3: iOS 15-17 — minimal consumer feature set

Test categories:
- rule accuracy
- false-positive rate
- battery drain
- memory footprint
- network latency
- extension stability

Device targets:
- Oldest supported iPhone
- Mid-gen devices
- Newest iPhone on iOS 26.x
- Supervised enterprise test device
