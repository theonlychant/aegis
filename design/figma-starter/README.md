Figma starter package

What this package contains:

- `tokens.json` (refer to the top-level design tokens) — colors, typography, spacing
- `components/` - SVG components you can paste or import into Figma: `button.svg`, `card.svg`, `badge-high.svg`, `badge-medium.svg`, `badge-low.svg`
- `mockups/` - `home-mockup.svg`, a simple Home screen mockup you can open or import into Figma

How to use in Figma:

1. Create a new Figma file.
2. Drag & drop the SVG files from this folder into the canvas. Figma will convert them into editable vectors/frames.
3. Use the tokens in `tokens.json` to set color styles, text styles, and spacing tokens.
4. Create components from the pasted SVGs to build a design system.

Notes:
- This starter is intentionally small: it's meant to seed a Figma workspace quickly.
- If you want, I can generate a .fig export if you provide Figma API access and a team/file to create under.
